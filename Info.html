🔧 Sobre a aplicação

A aplicação permite o gerenciamento de tarefas por usuários, incluindo criação e listagem de projetos, tarefas com prioridade fixa, regras de negócio como histórico de alterações, restrição de remoção de projetos com tarefas pendentes, limite de tarefas por projeto e geração de relatórios para perfis gerenciais.

O sistema está preparado para rodar via Docker e já possui o arquivo docker-compose.yml para facilitar a execução.

📦 Scripts do Banco

Os scripts de criação do banco estão incluídos, porém utilizei migrations para gerar automaticamente as estruturas. Caso precise, os scripts manuais estão disponíveis no repositório.

⚙️ Configuração de conexão com o banco

As configurações de conexão com o banco (endereço, usuário e senha) estão no arquivo:

appsettings.json (seção SqlServer)

Altere esses valores conforme seu ambiente local, se necessário.

▶️ Como rodar a aplicação

  1. Abra o terminal na pasta raiz do projeto, onde estão o .sln, Dockerfile e docker-compose.yml.
  2. Execute o comando abaixo para buildar e subir os containers:
docker-compose up --build

Aguarde o processo de build e inicialização. Você verá os logs da API e do SQL Server.

🌐 Acessando a API

Após os containers estarem rodando, acesse a documentação via Swagger:

http://localhost:8080/swagger

Qualquer dúvida, estou à disposição!
